COMMANDE
{
  numero: String (auto),
  client: { prenom, nom, email, telephone, adresse },
  items: [{produitId, name, price, taille, qty}],
  statut: 'en_attente' | 'validee' | 'en_route' | 'livree',
  total: Number
}

MESSAGE
{
  prenom, nom, email, telephone,
  sujet: String,
  message: String,
  lu: Boolean
}

CODE PROMO
{
  code: String (UPPERCASE),

  dateDebut, dateFin: Date
}

STOCK
{
  produitId: ObjectId,
  tailles: {S, M, L, XL, XXL: {quantite, minAlerte}},

}

🚀 DÉMARRAGE RAPIDE
════════════════════════════════════════════════════════════════

1. Installation dépendances
   npm install

2. Configuration .env
   MONGODB_URI=mongodb://localhost:27017/kyrace
   PORT=5000
   JWT_SECRET=your_secret
   ADMIN_EMAIL=admin@kyrace.com
   ADMIN_PASSWORD=admin123456

3. Initialiser données démo
   node scripts/seed.js

4. Démarrer serveur
   npm run dev

5. Accéder
   Frontend: http://localhost:5000
   Admin: http://localhost:5000/admin
   Credentials: admin@kyrace.com / admin123456
